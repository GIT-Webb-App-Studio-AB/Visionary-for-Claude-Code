# Palette Tokens - Cross-Style Color Registry

Auto-generated by `scripts/build-palette-tokens.mjs`. Lists every hex that
appears in the **Colors** section of 3 or more style files. Generators should
reference these via `var(--visionary-color-*)` instead of hardcoding the same
hex repeatedly.

OKLCH values are the perceptually-uniform equivalent (CSS Color 4, Baseline
2024). Prefer OKLCH for interpolation, color-mix(), and wide-gamut output.

## How to use in generated components

```css
:root {
  --visionary-color-anxiety-urgency: #FFFFFF; /* oklch(1.000 0.000 89.9) */
  --visionary-color-brutalist-honesty: #000000; /* oklch(0.000 0.000 0.0) */
  --visionary-color-legal-editorial: #1A1A1A; /* oklch(0.218 0.000 89.9) */
  --visionary-color-energetic-athletic: #0A0A0A; /* oklch(0.145 0.000 89.9) */
  --visionary-color-brutalist-honesty: #FF0000; /* oklch(0.628 0.258 29.2) */
}
```
