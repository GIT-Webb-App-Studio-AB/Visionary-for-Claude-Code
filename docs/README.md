# visionary-claude -- Marketplace Listing

**Visual design plugin for Claude Code by GIT Webb & App Studio AB.**

186 design styles. 8-step selection algorithm with weighted random. Closed visual feedback loop. Motion-first generation. Learns from rejection across sessions.

## What makes this different

| Feature | frontend-design | UI/UX Pro Max | visionary-claude |
|---------|----------------|---------------|-----------------|
| Design styles | ~15 implicit | 67 named | **186 with auto-inference** |
| Style selection | Manual/prompt | Manual name entry | **8-step algorithm with weighted random** |
| Anti-default bias | None | None | **Blocks generic AI output** |
| Motion quality | None | None | **Spring token system (motion/react)** |
| Visual feedback loop | None | None | **Playwright closed loop** |
| Taste memory | None | None | **system.md (learns across sessions)** |
| Accessibility | Not enforced | Not enforced | **WCAG 2.2 AA baked in** |
| i18n typography | ASCII only | ASCII only | **20+ languages with correct diacritics** |
| Component awareness | None | None | **Filters styles by UI type** |
| Determinism | Same in = same out | Same in = same out | **Weighted random from top 3** |
| Python required | No | Yes (bugs on Windows) | **No -- shell + Node only** |

## Quick start

```bash
claude plugin marketplace add GIT-Webb-App-Studio-AB/Visionary-for-Claude-Code
claude plugin install visionary-claude
```

Then describe any UI task -- the plugin activates automatically.
